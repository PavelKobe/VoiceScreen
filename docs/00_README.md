# AI Voice Screening — план запуска

Голосовой AI-агент для первичного скрининга кандидатов на массовых позициях (курьеры, кассиры, комплектовщики, продавцы, операторы). HR получает готовый шорт-лист вместо 200 входящих откликов в день.

## Ключевые цифры

| Параметр | Значение |
|---|---|
| Целевой MRR за 90 дней | $3 000–4 000 (≈ 270–360 тыс. ₽) |
| Средний чек (ARPU) | 15 000–30 000 ₽/мес |
| Нужно клиентов для цели | 10–15 |
| Бюджет на инфраструктуру | $50–150/мес на старте |
| Свободный бюджет | $500–700/мес на трафик и пилоты |

## Почему именно этот проект

- **Короткая дистанция до цели.** 10–15 B2B-контрактов реальнее за 3 месяца, чем 100+ SMB-подписок.
- **Острая и измеримая боль.** HR в массовом найме тонет физически. ROI клиент считает сам за минуту.
- **Слабая конкуренция в среднем ценовом сегменте.** Enterprise (Robot Vera) дорого, СМБ-ниша почти пуста.
- **Помещается в один VM.** Без парсеров, антиботов и серых зон.
- **Продажи через прямой outreach.** ЦА концентрированная и доступная.

## Стратегия в одну строку

Найти 3 пилотных клиента за 30 дней бесплатно/за символическую плату → отполировать продукт до стабильного 70%+ Call Completion Rate → перевести в платные + закрыть ещё 10–12 на втором-третьем месяце.
