# CLAUDE.md — app/bot/

Telegram-бот для HR-клиентов (основной интерфейс на MVP).

## Модули

- `__main__.py` — точка входа (`python -m app.bot`).
- `handlers.py` — обработчики команд:
  - `/start` — приветствие и меню.
  - `/register` — регистрация компании (создание Client).
  - `/upload` — загрузка CSV с кандидатами.
  - `/status` — текущий статус обзвона.
  - `/report` — получить отчёт по завершённым звонкам.

## Правила

- Используем `aiogram 3.x`.
- Бот связан с клиентом через `Client.tg_chat_id`.
- Уведомления о прошедших скрининг кандидатах отправляются автоматически после `finalize_call`.
